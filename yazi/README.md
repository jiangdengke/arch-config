# Yazi 快捷键说明

这份说明以当前仓库里的配置和本机安装的 `Yazi 26.1.22` 为准。

如果你从 shell 里启动，建议直接用 `y`，不要用 `yazi`：

```bash
y
```

这样退出后，shell 当前目录会自动跟随你在 Yazi 里最后停留的位置。

## 自定义快捷键

下面这些是这份配置里额外加的，或者覆盖了默认行为的键：

| 操作 | 快捷键 | 说明 |
| --- | --- | --- |
| 进入目录 / 打开文件 | `l` | 目录会进入；文件会直接按规则打开 |
| 打开挂载管理器 | `M` | 调用 `mount.yazi` 插件 |
| 智能粘贴 | `p` | 优先粘贴到当前悬停目录，否则粘贴到当前目录 |
| 放大图片预览 | `+` | 调用 `zoom.yazi` |
| 缩小图片预览 | `-` | 调用 `zoom.yazi` |
| 回到家目录 | `g` → `h` | `cd ~` |
| 进入 `~/.config` | `g` → `c` | `cd ~/.config` |
| 进入下载目录 | `g` → `d` | `cd ~/Downloads` |
| 进入图片目录 | `g` → `p` | `cd ~/Pictures` |
| 进入 Yazi 配置目录 | `g` → `y` | `cd ~/.config/yazi` |

## 常用默认快捷键

这些键没有被你当前配置改掉，仍然是 Yazi 默认行为里最常用的那批：

### 导航

| 操作 | 快捷键 |
| --- | --- |
| 上移 | `k` / `↑` |
| 下移 | `j` / `↓` |
| 返回上级目录 | `h` / `←` |
| 进入目录 / 打开选中文件 | `Enter` |
| 预览向上滚动 | `K` |
| 预览向下滚动 | `J` |
| 跳到顶部 | `g` → `g` |
| 跳到底部 | `G` |
| 通过 `fzf` 跳目录 / 定位文件 | `z` |
| 通过 `zoxide` 跳目录 | `Z` |
| 交互式输入目录 | `g` → `Space` |

### 选择与打开

| 操作 | 快捷键 |
| --- | --- |
| 切换当前项选中状态 | `Space` |
| 进入可视选择模式 | `v` |
| 进入反向可视选择模式 | `V` |
| 全选 | `Ctrl-a` |
| 反选 | `Ctrl-r` |
| 取消选择 | `Esc` |
| 打开选中文件 | `o` / `Enter` |
| 交互式选择打开方式 | `O` |
| 查看文件信息 | `Tab` |

### 文件操作

| 操作 | 快捷键 |
| --- | --- |
| 复制（yank copy） | `y` |
| 剪切（yank cut） | `x` |
| 覆盖粘贴 | `P` |
| 取消复制/剪切状态 | `Y` / `X` |
| 丢进回收站 | `d` |
| 永久删除 | `D` |
| 新建文件/目录 | `a` |
| 重命名 | `r` |
| 显示 / 隐藏隐藏文件 | `.` |
| 运行 shell 命令 | `;` |
| 运行阻塞式 shell 命令 | `:` |

## 复制路径

| 操作 | 快捷键 |
| --- | --- |
| 复制文件路径 | `c` → `c` |
| 复制目录路径 | `c` → `d` |
| 复制文件名 | `c` → `f` |

## 帮助

| 操作 | 快捷键 |
| --- | --- |
| 打开帮助 | `~` / `F1` |
| 退出 Yazi | `q` |
| 退出但不把目录同步回 shell | `Q` |

## 说明

- 当前配置里，`l` 和 `p` 都覆盖了默认行为。
- 当前配置里，`-` 已经改成“缩小图片预览”，不再是默认的“创建绝对路径符号链接”。
- 文本文件会优先走 `$EDITOR` 打开；其它文件默认走 `xdg-open`。
- Git 仓库状态会通过 `git.yazi` 插件显示在文件列表里。
